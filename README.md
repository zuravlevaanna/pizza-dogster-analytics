# ПиццаДогстер — Модуль аналитики

Модуль сбора и анализа данных о продажах сервиса доставки

## Структура репозитория
- **WebApp** — панель с отчётами и дашбордами
- **AnalyticsEngine** — движок расчёта метрик
- **ServerApi** — API для получения данных
- **Docs** — документация
