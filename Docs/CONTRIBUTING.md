# Как помочь проекту

## Создание Issue
Описывайте проблему или идею подробно

## Создание ветки
Формат: компонент/тип/описание
Пример: analytics/feature/revenue-calculation

## Создание Pull Request
- Название должно отражать суть изменений
- В описании укажите: Closes #номер_issue

## Правила
- Коммиты на русском или английском
- Одна задача = одна ветка = один PR
