# Требования к модулю аналитики ПиццаДогстер

## Функциональные требования
- Сбор данных о заказах (время, сумма, состав)
- Расчёт выручки за период (день/неделя/месяц)
- Определение самых популярных товаров
- Анализ времени доставки
- Отчёт по часам (пиковые часы)
- Экспорт отчётов в Excel/PDF

## Нефункциональные требования
- Быстрая загрузка отчётов (менее 3 секунд)
- Автообновление данных каждый час
- Защита данных (только администраторы)
